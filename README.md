# Online Quiz System

Backend-intensive quiz application built using:

- Node.js
- Express.js
- MySQL

## Features
- User registration
- Quiz creation (Admin)
- Questions & options storage
- Score calculation
- REST APIs

## How to Run

1. Install dependencies:

npm install


2. Setup .env file:

DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=quiz_app
PORT=5000


3. Start server:

npm run dev
