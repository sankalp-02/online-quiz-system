# Online Quiz Backend (MVP)

## Features
- User login (JWT)
- Create quiz
- Add questions with options
- Get quiz with questions

## Tech
- Node.js
- Express
- MySQL

## API Endpoints

POST /api/users/login  
POST /api/quiz/create  
POST /api/quiz/add-question  
GET /api/quiz/:quizId  

## How to run
1. npm install
2. setup .env
3. node server.js
